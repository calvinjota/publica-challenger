# Teste Prático - Pública

### Sobre

Aplicação feita na linguagem Kotlin, na plataforma Android.

### Como executar

__Requisitos__
- Java 8+
- Android Studio
- ADB

__Passo a passo__
- Abra no Android Studio o projeto em nível raiz
- Faça download do SDK do Android version 30
- E execute o projeto na IDE
